# random-bpm

An Ableton Live extension built with `@ableton-extensions/sdk`.

## Get Started

Download the latest version of the extension from [the releases page](https://github.com/benjamingoodheart/random-bpm)

## Installation

In Live, go to Settings > Extensions, then drag and drop the .ablx file into the window. Restart live.

![Install Demo](/.demos/1.install-random-bpm.gif)

## Usage

To randomize your BPM, right click an audio track, audio clip, midi track, midi clip, or scene and find the Extensions option in the menu. `Then click random-bpm: Randomize BPM option`

![Usage Demo](/.demos/2.randombpm-in-action.gif)
